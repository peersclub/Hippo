# ЁЯжЫ Hippo тАФ Vault Home

Conversational trading agent embedded in partner exchanges: **thin client, heavy core**. A one-script-tag surface that attaches to any website without harming it, deeply connected to the product underneath тАФ it understands the user and places orders from inside the conversation. Installation is agentic and CLI-based: `hippo init` understands the partner's site and generates the integration.
